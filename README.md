# termux-game
Install and play games by dclxviclan studio inside Termux terminal 

```
git clone https://github.com/vo6i/termux-game.git
cd termux-game
apt install ./dclxviclanSnak3.deb
```
# After installation package run in termux 
```
$ dclxviclanSnak3
```
